# Acceptance Criteria

## AC-1: Functional Fix
- [ ] Incorrect overdue markings are resolved
- [ ] Returned items are no longer listed overdue

## AC-2: Reporting Accuracy
- [ ] Overdue reports match source borrowing and return records
- [ ] Spot checks confirm data consistency

## AC-3: Documentation
- [ ] Root cause is documented
- [ ] Fix approach and validation steps are documented
