# User Stories

## Story 1: Reserve an Unavailable Book
As a library member,
I want to reserve a book when all copies are borrowed,
So that I can claim it when it becomes available.

**Acceptance Summary:**
- [ ] AC-1: Backend supports creating reservations for unavailable books with queue position assignment
- [ ] AC-2: Backend returns reservation queue data for a specific book
- [ ] AC-3: Books page UI shows a Reserve action only when no copies are available
- [ ] AC-4: Reservation flow returns clear success and duplicate-validation feedback
